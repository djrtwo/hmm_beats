\section{Instructed}

Perform the following actions:
\vskip1em
\noindent \bf{Sequence 1}

\begin{itemize}
    \item Turn on the sequencer
    \item Add a snare drum beat sequence
    \item Request the open high hat
    \item Edit the open high hat
    \item Request the closed high hat
    \item Edit the closed high hat
    \item Request the bass drum
    \item Make edits until you are satisfied with the beat
\end{itemize}

\noindent \bf{Sequence 2}
\begin{itemize}
    \item Clear each drum
    \item Request the closed high hat
    \item Edit the closed high hat
    \item Request the bass drum
    \item Request the open high hat
    \item Request the snare drum
    \item Clear the bass drum
    \item Edit the snare drum
    \item Request the bass drum
    \item Edit the snare drum
    \item Request the bass drum
    \item Edit the open and closed high hats
    \item Request the bass drum
\end{itemize}

\section{Open Instructed}

Take 1 minute to create an electronic beat utilizing the following sequence:

\begin{itemize}
    \item Turn on the sequencer
    \item Add a beat sequence for two drums
    \item Request an additional drum
    \item Edit any of the three drum sequences
    \item Request a new sequence for one of the three drums
    \item Edit any of the three drum sequences
    \item Request the fourth drum
    \item Clear two of any of the four drum sequences
    \item Add a drum sequence for one of the cleared drums
    \item Request the last drum
\end{itemize}
